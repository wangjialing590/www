# 校园智能排课系统

基于遗传算法的智能排课系统，使用Flask框架实现前后端交互。

## 环境要求

- Python 3.9+

## 安装命令

```bash
pip install -r requirements.txt
```

## 依赖包

- flask==3.0.0
- pandas==2.1.4
- openpyxl==3.1.2
- numpy==1.26.2

## 运行命令

```bash
python app.py
```

访问 http://localhost:5000 即可使用系统。

## 使用步骤

### 1. 导入数据

在首页拖拽或选择包含排课数据的Excel文件上传。Excel文件必须包含以下5个工作表：

- **教师信息表**：教师姓名、工号、所属院系、住宿情况、授课偏好、联系电话、不可授课时段
- **班级信息表**：班级名称、所属专业、年级、学生人数
- **教室信息表**：教室ID、座位数、教室类型
- **课程信息表**：课程ID、课程名称、学分、周学时、总学时、连排要求、教室类型要求、是否允许晚间、是否隔周上课
- **教学任务表**：任务ID、课程、授课教师、授课班级、周学时、连排要求、教室类型要求、学期、状态

### 2. 开始排课

点击"开始排课"按钮启动遗传算法进行排课。系统会实时显示排课进度和当前最佳适应度。

### 3. 查看课表

排课完成后，点击"查看课表"进入课表页面。可以按教师、班级或教室查看对应的课表。

### 4. 导出结果

在排课页面点击"导出课表"按钮，可以将排课结果导出为Excel文件。

## 排课规则

### 硬约束（必须满足）

1. 同一教师同一时间只能有一门课
2. 同一教室同一时间只能有一门课
3. 同一班级同一时间只能有一门课
4. 住宿情况为"住家里"的教师不能安排晚间课程（9-11节）
5. "是否允许晚间"为"否"的课程不能安排晚间课程（9-11节）
6. 教室座位数必须大于等于班级学生人数
7. 连排课程必须按要求连排（2学分课2节连排，4学分课白天4节连排/晚间3节连排）
8. 不能安排教师不可授课时段

### 软约束（优化目标）

1. 教师授课偏好匹配
2. 课程分布均匀
3. 教室类型匹配
4. 时间分布合理
